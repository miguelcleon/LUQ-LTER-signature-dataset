How to Use the Luquillo LTER Signature Dataset Explorer

Welcome to the Luquillo Long-Term Ecological Research (LTER) Signature Dataset Explorer! This Shiny app allows you to explore and visualize long-term ecological measurements from the Luquillo Experimental Forest in Puerto Rico.

Getting Started

1. Selecting Data Frequency

In the sidebar panel on the left, you'll find the Select data frequency options:

  • Daily: Use the daily dataset.
  • Monthly: Use the monthly dataset.

Select the desired data frequency to load the corresponding dataset.

2. Choosing Variables to Plot

Under Pick one or more variables to plot, select one or more variables from the dropdown list. These variables will be used for plotting and data analysis.

You can select multiple variables by holding down the Ctrl (Windows) or Command (Mac) key while clicking on the variable names.

3. Normalizing Data

If you wish to normalize the selected variables between 0 and 1, check the Normalize values between 0 and 1 checkbox.

Normalization can help in comparing variables that are on different scales.

4. Plotting the Data

Click the Plot the data button to generate the plot based on your selections.

The plot will display in the main panel on the right. You can interact with the plot by hovering over data points to see detailed information.

5. Viewing Data Summary

Below the plot, a data summary is displayed. It provides statistical information about the selected variables.

6. Downloading the Data

To download the processed data, click the Download Data button. The downloaded CSV file will include:

  • The Date column.
  • The selected variables.
  • Data starting from the first date where data is available for the selected variables.

Understanding the App Features

Data Cleaning

The app automatically handles missing values and zeros in the data:

  • Zeros in numeric columns are replaced with NA.
  • Leading rows where all selected variables are NA are removed.

Date Parsing

The app parses dates from the selected dataset and ensures they are in a consistent format for plotting and analysis.

Interactive Plotting

The plot generated is interactive, thanks to Plotly integration:

  • Hover over data points to see tooltips with detailed information.
  • Legend entries allow you to toggle the visibility of variables.
  • You can zoom in and out using the toolbar or by selecting an area on the plot.

Troubleshooting

Note:

If you encounter any issues while using the app, consider the following tip:

  • If the plot does not display, verify that you have selected at least one variable and clicked the Plot the data button.

About the Dataset

This application uses data from the Luquillo Long-Term Ecological Research (LTER) Signature Dataset. The dataset contains long-term ecological measurements from the Luquillo Experimental Forest in Puerto Rico.

The Luquillo LTER program focuses on understanding the effects of disturbances (such as hurricanes and land-use changes) on tropical forest ecosystems. The signature dataset includes a variety of ecological variables measured over extended periods.
